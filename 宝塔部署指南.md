# 🚀 宝塔面板部署学号查询系统指南

## 📋 部署前准备

### 1. 服务器要求
- **操作系统**: Ubuntu 22.04 LTS
- **内存**: 至少 1GB RAM
- **硬盘**: 至少 10GB 可用空间
- **网络**: 公网IP或域名

### 2. 安装宝塔面板

#### 方法一：官方一键安装脚本
```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装宝塔面板
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
```

#### 方法二：手动安装
```bash
# 安装依赖
sudo apt install -y wget curl

# 下载并安装宝塔面板
curl -sSO https://download.bt.cn/install/install_panel.sh && sudo bash install_panel.sh
```

## 🔧 宝塔面板配置

安装完成后，记录显示的：

## 📁 部署学号查询系统

### 1. 创建网站目录
```bash
# 在宝塔面板 → 网站 → 添加站点
# 或者手动创建目录
sudo mkdir -p /www/wwwroot/student-query
sudo chown -R www:www /www/wwwroot/student-query
```

### 2. 上传项目文件

#### 方法一：通过宝塔文件管理器
1. 登录宝塔面板
2. 点击 **文件** → 进入 `/www/wwwroot/student-query`
3. 上传以下文件：
   - `index.html`
   - `style.css`
   - `script.js`
   - `students.csv`
   - `server.py` (可选)

#### 方法二：使用SCP命令
```bash
# 在本地电脑执行（替换your-server-ip为实际IP）
scp -r * root@your-server-ip:/www/wwwroot/student-query/
```

#### 方法三：使用Git
```bash
# 在服务器上执行
cd /www/wwwroot/student-query
git clone <你的仓库地址> .
# 或者直接下载文件
```

### 3. 配置Nginx网站

#### 在宝塔面板中配置：
1. **网站** → **添加站点**
2. 填写信息：
   - **域名**: 你的域名或IP
   - **根目录**: `/www/wwwroot/student-query`
   - **PHP版本**: 纯静态（或选择PHP版本）

#### 手动配置Nginx（可选）：
```nginx
server {
    listen 80;
    server_name your-domain.com;  # 替换为你的域名
    root /www/wwwroot/student-query;
    index index.html;
    
    # 启用gzip压缩
    gzip on;
    gzip_types text/css application/javascript text/javascript application/json;
    
    # 静态文件缓存
    location ~* \.(css|js|png|jpg|jpeg|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    # CSV文件特殊处理
    location ~ \.csv$ {
        add_header Access-Control-Allow-Origin *;
        add_header Content-Type "text/csv; charset=utf-8";
    }
    
    # 主页面
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 🔒 安全配置

### 1. 防火墙设置
```bash
# 开放必要端口
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
sudo ufw allow 8888/tcp  # 宝塔面板
sudo ufw enable
```

### 2. SSL证书配置（推荐）
在宝塔面板中：
1. **网站** → **你的站点** → **SSL**
2. 选择 **Let's Encrypt** 免费证书
3. 点击 **申请** 并 **强制HTTPS**

### 3. 访问限制
- 修改宝塔面板默认端口（8888 → 自定义端口）
- 设置面板访问白名单IP
- 定期更新宝塔面板版本

## 📊 数据管理

### 1. 更新学生数据
```bash
# 方法一：通过宝塔文件管理器
# 直接编辑 /www/wwwroot/student-query/students.csv

# 方法二：命令行编辑

# 方法三：上传新文件覆盖
```

### 2. 数据备份
```bash
# 创建备份脚本
sudo nano /root/backup_student_data.sh
```

备份脚本内容：
```bash
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/root/backups"
SOURCE_DIR="/www/wwwroot/student-query"

mkdir -p $BACKUP_DIR
tar -czf $BACKUP_DIR/student-query_$DATE.tar.gz -C $SOURCE_DIR .
echo "备份完成: $BACKUP_DIR/student-query_$DATE.tar.gz"

# 保留最近7天的备份
find $BACKUP_DIR -name "student-query_*.tar.gz" -mtime +7 -delete
```

设置定时备份：
```bash
sudo chmod +x /root/backup_student_data.sh
sudo crontab -e
# 添加：每天凌晨2点备份
0 2 * * * /root/backup_student_data.sh
```

## 🆘 常见问题解决

### 1. 无法访问网站
```bash
# 检查Nginx状态
sudo systemctl status nginx

# 重启Nginx
sudo systemctl restart nginx

# 检查端口占用
sudo netstat -tlnp | grep :80
```

### 2. CSV文件无法加载
- 检查文件权限：`sudo chmod 644 /www/wwwroot/student-query/students.csv`
- 检查文件编码：确保为UTF-8
- 检查Nginx配置中的CORS设置

### 3. 宝塔面板无法访问
```bash
# 重启宝塔服务
sudo /etc/init.d/bt restart

# 查看宝塔状态
sudo /etc/init.d/bt status

# 重置宝塔密码
sudo bt default
```
